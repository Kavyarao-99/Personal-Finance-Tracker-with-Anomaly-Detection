## Overview
This project is a personal finance tracker built with Python and Streamlit. 
It allows users to track income, expenses, and savings, visualize financial trends, and detect unusual transactions using statistical (Z-score) and machine learning (Isolation Forest) methods.

## Features
- Income and expense tracking  
- Interactive visualizations using Plotly and Seaborn  
- Anomaly detection to flag unusual transactions  
- Synthetic dataset generation for testing

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn, Plotly
- Streamlit
